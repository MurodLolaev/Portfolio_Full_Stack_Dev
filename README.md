# 🚀 Моё Личное Портфолио / Сайт-визитка

Стильный, современный одностраничный сайт-портфолио с адаптивным дизайном в стиле **Glassmorphism** (матовое стекло), яркими неоновыми акцентами и интерактивными элементами.

## 🛠️ Технологический стек

*   **HTML5** — семантическая структура сайта.
*   **Tailwind CSS** — стилизация, анимации, адаптивная сетка (Grid/Flexbox).
*   **JavaScript** — интерактивность (плавные переходы, логика интерфейса).

## ✨ Особенности проекта

*   **Glassmorphic Design:** Полупрозрачные карточки с размытием заднего фона (`backdrop-blur`).
*   **Светлая / Темная тема:** Интерфейс адаптирован под современные тренды веб-дизайна.
*   **Полный адаптив:** Сайт идеально отображается на смартфонах, планшетах и компьютерах.
*   **Интерактивные контакты:** Живые hover-эффекты со свечением элементов при наведении.
